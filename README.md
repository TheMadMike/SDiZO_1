# SDiZO_1 - project for university Data Structures course

# Building and running (GNU/Linux)

## Dependencies:

- CMake 3.x
- g++ 11.x
- GNU Make

## Build

In order to build the project type:
```
  mkdir build
  cd build
  cmake ..
  make
```

## Run

Run the app:
```
  ./sdizo
```

## Run tests

```
  ./sdizo/src/tests/test
```
